# 🎬 Netflix Movies & TV Shows Analysis (EDA)

## 📌 Project Overview
This project was completed as part of the **Pluto Academy AI & Machine Learning Internship Program**.

The objective of this project is to perform **Exploratory Data Analysis (EDA)** on the Netflix Movies & TV Shows dataset to discover trends, patterns, and insights using Python. The project includes data cleaning, visualization, and business recommendations based on the analysis.

---

## 📂 Dataset

- **Dataset:** Netflix Movies & TV Shows
- **Source:** Kaggle
- **Link:** https://www.kaggle.com/datasets/shivamb/netflix-shows

---

## 🎯 Project Objectives

- Load and inspect the dataset
- Clean missing and duplicate values
- Analyze movies and TV shows
- Explore trends over time
- Visualize important patterns
- Generate business insights and recommendations

## 🔍 Key Insights

Some important findings from the analysis include:

- Netflix contains both Movies and TV Shows, with Movies being more common.
- Content production has increased significantly over recent years.
- Drama, Comedy, and International content dominate the platform.
- The United States contributes the largest amount of Netflix content.
- Netflix has expanded its global content library across multiple countries.

---

## 💼 Business Recommendations

- Increase investment in high-demand genres.
- Expand regional content production.
- Continue focusing on international audiences.
- Promote newly released content more effectively.
- Improve recommendations using genre and country trends.
